## 🛰️ Projelerimiz

AstroBotix çatısı altında, farklı görev ortamları için beş ayrı otonom platform geliştiriyoruz:

| Proje | Platform |
|---|---|
| **PLANET UAV** | Otonom hava aracı |
| **PLANET ROVER** | Otonom kara aracı |
| **STINGRAY** | Otonom deniz/su altı aracı |
| **TAYA** | Otonom araç |
| **WALRUS19** | Otonom araç |

Her proje kendi göreve özel donanım ve sürüş mantığına sahip olsa da, hepsi **aynı yazılım çekirdeği** üzerine inşa ediliyor.
